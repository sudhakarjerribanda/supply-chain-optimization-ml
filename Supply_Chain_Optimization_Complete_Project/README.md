# Supply Chain Optimization using Machine Learning

This project analyzes real-world supply chain datasets and applies machine learning models for forecasting demand, lead-time behavior, and operational optimization.

## Project Structure
- data/: Raw datasets
- notebooks/: Analysis, feature engineering, forecasting, and results
- src/: Python modules
- reports/: Abstracts, literature surveys, related work
- images/: Figures
- references/: Supporting references
- presentations/: PPTs and documents

## How to Run
Install dependencies:
```
pip install -r requirements.txt
```
Then open notebooks:
```
jupyter notebook notebooks/
```
